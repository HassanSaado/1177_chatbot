Tekniska krav för att logga in på 1177.se
För att kunna logga in på 1177.se behöver du använda dig av de operativsystem och webbläsare som rekommenderas nedan samt aktivera JavaScript för vald webbläsare. Du behöver också ha en e-legitimation.
Operativsystem är det program som du styr telefonen, surfplattan eller datorn med. Webbläsare är det program som du använder för att läsa webbsidor på internet.
När du använder mobiltelefon eller surfplatta
Du kan använda den webbläsare som finns i telefonen eller surfplattan om du har någon av följande:
Iphone med en uppdaterad version av iOS
Ipad med en uppdaterad version av iOS
Telefoner och surfplattor med en uppdaterad version av android.
När du använder dator
De operativsystem för datorer som rekommenderas är:
Windows 10 eller nyare
MacOS 10.15 eller nyare.
Webbläsare för datorer
För PC rekommenderas följande webbläsare:
Google Chrome, senaste versionenLadda ner Google Chrome
Microsoft Edge, senaste versionenLadda ner Microsoft Edge.
För Mac rekommenderas följande webbläsare:
Safari, senaste versionen.
Google Chrome, senaste versionen.Ladda ner Google Chrome.
Andrawebbläsare eller operativsystem
E-tjänsterna kan fungera med andra webbläsare eller operativsystem. Det finns dock ingen garanti att de fungerar som de ska.
Du kan heller inte få support vid eventuella problem om du använder andra webbläsare eller operativsystem än de rekommenderade.
Aktivera JavaScript
För att e-tjänsterna ska fungera behöver JavaScript aktiveras i webbläsaren. JavaScript tillåter webbsidans fullständiga funktioner.
På skatteverket.se kan läsa instruktioner om hur du aktiverar JavaScript
Du måste ha rätt språkinställningar
När du loggar in i journalen behöver du ha språket inställt på svenska. Annars finns risk att din journal översätts felaktigt.
Du behöver en e-legitimation för att logga in på 1177.se
För att logga in på 1177.se behöver du ha ett svenskt personnummer och en e-legitimation, antingen mobilt bank-id, bank-id på kort, bank-id på fil eller Freja+.
Du kan inte logga in på 1177.se med bankernas mobilt säkerhets-id eller motsvarande e-legitimationer med lägre säkerhet.
För tillfället kan du inte logga in på 1177.se med e-legitimation från Svenska pass. Det är den e-legitimation som finns på Skatteverkets id-kort, utfärdade från september 2017.

Det här kan du göra när du loggat in
