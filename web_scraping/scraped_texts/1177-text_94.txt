Att få stöd, behandling och utbildning via nätet
Ibland kan du få det stöd, den behandling eller utbildning som du behöver via nätet. Det kan till exempel vara ett behandlingsprogram där det ingår övningar som du gör själv.
Behandlingsprogrammet kan bland annat innebära att du ska läsa texter, svara på frågor, titta på filmklipp eller lyssna på ljudfiler. Det kan också vara att du ska göra övningar. Det varierar vad som ingår. Du kan göra övningarna hemma eller på någon annan plats där det känns bra.
Hur lång tid det tar att gå igenom ett program varierar och beror på vad det är för typ av stöd, behandling eller utbildning.
Hur går det till?
Det finns olika sätt att gå ett behandlingsprogram via nätet.
Du kan få en inbjudan till ett program av en behandlare. Hen finns då oftast med som stöd och kan ge dig återkoppling och svara på dina frågor.
Du kan också själv påbörja ett program som en region erbjuder. Det kan betyda att du har kontakt med behandlare, men det kan också vara ett självhjälpsprogram. Då är du själv aktiv och följer programmet en viss tid, vanligtvis några månader.
Programmen kan användas enskilt eller som ett komplement till annan behandling.
Många olika typer av program
Det finns många olika typer av program för stöd, behandling och utbildning via nätet. Det skiljer sig åt mellan regionerna vad som erbjuds. Alla mottagningar erbjuder inte den här typen av stöd, behandling eller utbildning.
Hur mycket det kostar varierar mellan regioner.
De flesta regioner erbjuder KBT via nätet.Här kan du läsa mer om hur det går till att få KBT via nätet.

Om 1177
1177  är hela Sveriges samlingsplats för information och tjänster inom hälsa och vård. Vi erbjuder sjukvårdsrådgivning, information, inspiration och e-tjänster. Dygnet runt finns vi på 1177.se och på telefon 1177 för sjukvårdsrådgivning.
Läs din journal via nätet
Du som är 16 år eller äldre kan läsa information i din journal genom att logga in. Hur mycket information du kan läsa beror på i vilken region du har fått vård.
Boka, omboka eller avboka en tid
Passar inte tiden? Genom att logga in på 1177.se kan du till exempel boka, omboka eller avboka en tid.
1177
–tryggt om din hälsa och vård
På 1177.se får du råd om hälsa och information om sjukdomar och vilka mottagningar du kan kontakta. Logga in för att läsa din journal och göra dina vårdärenden. Ring telefonnummer 1177 för sjukvårdsrådgivning dygnet runt.1177 ger dig råd när du vill må bättre.
