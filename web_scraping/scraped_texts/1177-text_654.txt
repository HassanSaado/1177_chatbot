Hur får jag bort sår i näsan?
Fråga
Jag har sår i näsan. Finns det något jag kan göra för att få bort det?
Svar
Det är vanligt med sår eller sårskorpor i näsan. Det kan leda till att detblödereller luktar illa i näsan. Såren orsakas oftast av en infektion med virus eller bakterier och brukar försvinna av sig själv.
Försök att inte pilla på såren, även om det kan vara svårt. Annars kan bakterier spridas och då tar det längre tid för slemhinnan i näsan att läka.
Det kan du göra själv
Du kan använda koksaltlösning för att mjuka upp sårskorporna så att de lättare kommer ut. Det finns koksaltdroppar och koksaltsprej att köpa på apotek.
Du kan använda lite matolja för att mjuka upp slemhinnorna i näsan. Det finns också mjukgörande nässprej och nässalva att köpa på apotek.
Ibland behövs behandling
Du kan behöva behandling om såren inte läker. Ring telefonnummer 1177 om du vill hasjukvårdsrådgivning. Då kan du få hjälp att bedöma symtom eller hjälp med var du kan söka vård.
