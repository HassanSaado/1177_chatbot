Vaccination mot covid-19 och influensa
Vaccination mot influensa och covid-19
Nu går det att vaccinera sig mot influensa och covid-19. Se till att få de vaccindoser du rekommenderas. Det ger ett bra skydd inför vintern när många brukar bli sjuka.
I de flesta regioner kommer du att kunna vaccinera dig mot influensa och covid-19 vid samma tillfälle.
Välj region högst upp på sidan för att se vad som gäller där du är.
När kan jag vaccinera mig?
Se till att få vaccin nu om du rekommenderas att vaccinera dig. Det finns uppdaterat vaccin som skyddar även mot nyare virusvarianter.
Behöver jag vaccinera mig?
Här kan du läsa mer om vem som rekommenderas att vaccinera sig.

Kan jag vaccinera mig mot fler sjukdomar samtidigt?
Det går att vaccinera sig mot flera sjukdomar vid samma tillfälle.
En del kan få vaccin mot till exempelpneumokockersamtidigt som de får vaccin mot influensa och covid-19.
Det beror bland annat på hur du mår och vilken sorts vaccin du får om du kan vaccinera dig mot två eller flera sjukdomar samtidigt. Den som vaccinerar dig bedömer vad som blir bäst för dig.

Det här kan du göra när du loggat in
Du kan till exempel förnya recept och boka, omboka eller avboka tider. Olika mottagningar erbjuder olika tjänster.
Så tvättar du händerna
Många sjukdomar kan spridas genom kontakt med händer. Genom att tvätta händerna kan du undvika att själv bli sjuk. Du kan också undvika att smitta andra om du redan är sjuk.
