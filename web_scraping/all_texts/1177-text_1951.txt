Bröstmjölksersättning och tillskottsnäringBröstmjölksersättning kan helt eller delvis ersätta bröstmjölk under barnets första halvår. Ersättningarna är gjorda för att likna bröstmjölk så mycket som möjligt och innehåller all näring barnet behöver. Från sex månader kan barnet börja dricka tillskottsnäring istället, som ett komplement till vanlig mat.
Bröstmjölksersättning och tillskottsnäring
Bröstmjölksersättning kan helt eller delvis ersätta bröstmjölk under barnets första halvår. Ersättningarna är gjorda för att likna bröstmjölk så mycket som möjligt och innehåller all näring barnet behöver. Från sex månader kan barnet börja dricka tillskottsnäring istället, som ett komplement till vanlig mat.

När du inte kan eller vill ammaIbland fungerar inte amningen och ibland är den aldrig ett alternativ. Barnet får då urpumpad bröstmjölk eller bröstmjölksersättning istället. Det är viktigt att inte känna skuld om du inte kan eller vill amma.
